En conclusión, establecemos que el máximo número de usuarios que puede llegar a registrar nuestra applicación es de 6200 usuarios.

Para realizar estas pruebas hemos tenido que hacer uso de un csv, ya que, el nombre de usuario y el email de un
actor deben ser único. Para general el csv hemos utilizado un pequeño programa que se encuentar en la carpeta 
generateUserNameEmail y los diferentes archivos generados para hacer las pruebas se encuentran en archivos generados.

En cada captura se añadio también una foto de mySQL indicando que se han creado los usuarios correctamente en cada prueba,
por ejemplo en la primera prueba se crearon 1500 usuarios más el que ya existía en la base de datos (el usuario, que se crea al hacer
la prueba de elimino antes de lanzar las pruebas)
